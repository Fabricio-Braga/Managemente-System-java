# Member Management System

A simple Java console application built to manage member data and practice Object-Oriented Programming (OOP) fundamentals.

## What it does??
- **Add & Remove Members:** Register new profiles with validation (blocks invalid ages or negative values).
- **Data Persistence:** Saves and loads all data automatically to a local `socios.txt` file using Java I/O (`PrintWriter` and `Scanner`).
- **Search & Reports:** Search members by name and calculate analytics like average age and unpaid balances.
- **Business Rules:** Automatically applies a 5% fee for late members or a 9% discount based on specific age and pricing criteria.
